# Daily Focus Timer

A plain HTML/CSS/JS web application to track your focus sessions.

## Features
- **Customizable Timer**: Set your focus duration (1-180 minutes).
- **Daily Progress**: Tracks how many sessions you complete today (resets automatically at midnight).
- **Offline Capable**: Standard web technologies, no backend required.
- **Premium UI**: Glassmorphism design and smooth animations.

## How to Run Locally

1. Open this folder in your browser specific file viewer, or just double-click `index.html`.
2. The app works immediately without any installation.

## How to Deploy to GitHub Pages

1. **Create a GitHub Repository**:
   - Go to [github.com/new](https://github.com/new).
   - Name it `daily-focus-timer` (or anything you like).
   - Make sure "Public" is selected.
   - Click "Create repository".

2. **Upload Files**:
   - In your new repository, click "uploading an existing file".
   - Drag and drop `index.html`, `style.css`, and `script.js` from this folder.
   - Commit the changes.

3. **Enable GitHub Pages**:
   - Go to the repository **Settings**.
   - Click **Pages** in the left sidebar.
   - Under "Build and deployment", select **main** branch as the source.
   - Click **Save**.

4. **Get Your Link**:
   - Refresh the page after a minute.
   - You will see your deployed link (e.g., `https://yourusername.github.io/daily-focus-timer/`).

## Project Structure
- `index.html`: The main structure.
- `style.css`: Visual styling (colors, layout, animations).
- `script.js`: Application logic (timer, storage, events).
